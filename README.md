## Requirements

- Nodejs v16
- npm v6.14

## Quick start

- Clone the repo: `git clone ${url}`
- Open root folder : `cd seller-management`
- Install Depedencies: `npm install`
- Run App Dev: `npm start`
- Build App : `npm run-script build`
